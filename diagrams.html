Nganya Fleet Management System Documentation

System Overview

The Nganya Fleet Management System is documented through a series of diagrams that provide a comprehensive view of the system's architecture and functionality. The Use Case Diagram illustrates the system's core functionalities and user interactions, showing how administrators and drivers interact with features like fleet management, route planning, and scheduling. The Activity Diagram details the route planning process, including weather checks, traffic monitoring, and driver assignments, providing a clear workflow of how routes are managed. The Swimlane Diagram breaks down the system interactions between administrators, drivers, and external services, showing the flow of information and responsibilities across different system components. The Sequence Diagram focuses on the driver assignment process, demonstrating the step-by-step interaction between the administrator, system, driver, and weather service. Finally, the Class Diagram provides a detailed view of the system's data structure, showing the relationships between key entities like users, drivers, vehicles, routes, and maintenance records. Together, these diagrams offer a complete picture of how the system operates, from high-level user interactions to detailed data relationships.

Database Schema Description

Table Name Description Key Relationships
USERS Stores user authentication and profile information Parent table for DRIVERS
DRIVERS Contains driver-specific information and credentials Links to USERS, SCHEDULES, and DRIVER_ASSIGNMENTS
VEHICLES Stores fleet vehicle information and status Links to SCHEDULES, MAINTENANCE, and DRIVER_ASSIGNMENTS
ROUTES Contains route information and waypoints Links to SCHEDULES, WAYPOINTS, and WEATHER_ALERTS
SCHEDULES Manages route assignments and timing Links to ROUTES, DRIVERS, and VEHICLES
WAYPOINTS Stores intermediate points for routes Links to ROUTES
MAINTENANCE Tracks vehicle maintenance history Links to VEHICLES
WEATHER_ALERTS Stores weather-related alerts for routes Links to ROUTES
DRIVER_ASSIGNMENTS Manages driver-vehicle assignments Links to DRIVERS and VEHICLES
